# FPGAClustering
This is a simple Branch and Bound Partitioning algorithm.

## How to run
First clone repo:
`git clone https://github.com/EthanPan233/FPGAClustering.git`
First build
`mkdir build`
`cd build`
`cmake ../src`
`cmake --build .`
Then run program on benchmarks
`./Clustering <benchmark-name>`
For example:
`./Clustering cc`
